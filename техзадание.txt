Задание на разработку telegram бота (25.08.2023)
    *Фреймворк aiogram*


Глоссарий:
 - Пользователь - действующий или потенциальный клиент компании AVILINE
 - Бот - автоматизированная система в мессенджере телеграм
 -

Общее. Бот должен:
 - Предоставлять круглосуточный доступ пользователей / клиентов AVILINE к информации о продукции компании;
 - Позволять пользователям получить информацию на основные вопросы касаемо пробретения, установки и решения (основных?) возникающих проблем;
 - Предоставлять возможность пользователю комфортным для него образом оставить обратную связь о продукции;
 - Предоставлять возможность свзязаться с техподдержкой или менеджером по продажам напрямую через форму обратной связи внутри UI телеграм;
 - Давать возможность редактировать продукцию, её описание и прочую связанную информацию через админский функционал;

Реализация:
 - aiogram фреймоворк для обработки запросов к боту;
 - django для реализации админки;
 - aiosqlite / sqlalchemy + postgresql для реализации хранимых данных;
 - redis / aioredis / mongo для реализации быстрой доступности к данным.
